# ⏱️ FlowTime – Productivity App Website

FlowTime is a modern productivity app landing page built with **HTML, CSS, and JavaScript**.  
It helps users stay organized, track their tasks, and focus better using a clean and responsive interface.  

🌐 **Live Demo:** [View on Netlify](#)  

---

## 🚀 Features
- ✅ Task Manager – keep all your tasks in one place  
- ✅ Focus Timer – Pomodoro-style sessions to boost productivity  
- ✅ Insights & Analytics – track your progress and habits  
- ✅ Custom Themes – dark, light, or neon mode  
- ✅ Testimonials Carousel – feedback from real users
